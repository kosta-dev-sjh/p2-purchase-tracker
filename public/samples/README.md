# 샘플 파일

MVP 시연용 카드 이용내역 샘플입니다. `/csv-upload` 화면에서 그대로 업로드하면 동작합니다.

## 지원 파일 형식

- **CSV** (`.csv`): UTF-8 인코딩 권장 (BOM 자동 처리)
- **XLSX** (`.xlsx`, `.xls`): 상단 안내 행이 있어도 헤더 자동 감지
  - `xlsx` 라이브러리는 업로드 시점에 동적 로딩되므로 CSV만 쓰는 사용자는 받지 않습니다.

## 포맷 가이드

헤더: `이용일, 가맹점명, 이용금액, [카테고리]`

### 날짜
`YYYY-MM-DD`, `YYYY.MM.DD`, `YYYY/MM/DD`, `YYYYMMDD` 모두 허용

### 가맹점명
쿠팡/네이버/무신사 표기의 다양한 변형을 자동 정규화
(예: "쿠팡(주)", "COUPANG", "네이버페이", "N-Pay", "무신사", "MUSINSA")
그 외 가맹점은 업로드를 실패시키지 않고 플랫폼 `미지정`으로 저장

### 이용금액
쉼표와 "원" 기호, "KRW" 표기 자동 제거 후 숫자 파싱

### 카테고리(선택)
생활용품/패션/전자기기/식품/기타 중 하나. 없거나 미일치 시 "기타"로 기본 매핑

### 헤더명 변형 수용
아래 컬럼명은 모두 자동으로 인식됩니다.

| 의미 | 허용 헤더 |
| --- | --- |
| 날짜 | 이용일, 거래일, 결제일, 승인일, date |
| 가맹점 | 가맹점명, 가맹점, 사용처, merchant |
| 금액 | 이용금액, 승인금액, 결제금액, 금액, amount |
| 카테고리 | 카테고리, category |

## 주의 사항

- 날짜/금액/가맹점명이 비어 있거나 형식을 읽을 수 없는 행만 "건너뛴 행"으로 표시됩니다.
- XLSX의 경우 첫 번째 시트만 읽습니다.
- 카드사 엑셀 파일 상단에 `"홍길동님의 이용내역"` 같은 안내 문구가 1~5행 들어가도 자동으로 건너뜁니다.
